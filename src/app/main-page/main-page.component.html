Description

This Board Game Manager lets users log in to create and join groups and record games they have played in that group and record winners

This project is my largest solo project to date, it is a full stack program using sn SQL server, ASP.NET Core API and Angular front end using Material UI as well.

Technologies used

  • ASP.NET Core
  • C#
  • Angular
  • Html/Css/Typescript/Javascript
  • Sql Server
  • GitHub
  • Azure
  • Auth0

Difficulties

This project had some interesting challenges, the addition of a log in feature allows for lots of unique functionality, however, this also added complexity in ensuring the safety of users data. This was ensured by only storing non sensitive information on my server and relying on the trusted Auth0 service.
